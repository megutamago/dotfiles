# dotfiles

## Procedure

```bash
cd $HOME && \
git clone https://github.com/megutamago/dotfiles.git && \
cd dotfiles && \
chmod +x .bin/install.sh && \
./.bin/install.sh
```

```bash
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
git config --global credential.helper store
```
